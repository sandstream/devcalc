# DevCalc

A modern, keyboard-first web calculator for developers with instant hex/dec/bin/oct conversion, unix timestamp handling, and bitwise operations.

## Tech Stack

- React 18 + Vite
- TypeScript
- Tailwind CSS
- Vitest (unit) + Playwright (E2E)

## Development

```bash
npm install          # Install dependencies
npm run dev          # Start dev server
npm run build        # Production build
npm run test         # Run unit tests
npm run test:e2e     # Run E2E tests
```
