# DevCalc - Project Brief

## The Idea (One-liner)
En modern, keyboard-first webkalkylator för utvecklare med hex/bin/dec, timestamps, och history - designad för 2024, inte 2010.

## Core Motivation (from 5 Whys)
Klassiska kalkylatorer finns överallt, men developer-fokuserade verktyg har ofta daterad UI och saknar modern UX. Det finns en möjlighet att bygga något som utvecklare faktiskt *vill* använda dagligen - snabbt, snyggt, och med rätt features.

## The Hook / Differentiator
**"Spotlight för utvecklare"** - Live results utan "=" knapp, modern keyboard-first design, och history som faktiskt är användbar. Kombinerar det bästa från Raycast/Alfred-stilen med developer-specifika beräkningar.

Konkurrenternas gap:
- De flesta dev-kalkylatorer ser ut som Windows XP
- Webb-alternativ är få och basic
- Ingen har keyboard shortcuts + modern UI + persistent history

## Target Audience
**Primary:** Utvecklare som regelbundet jobbar med:
- Hex/binary (färger, bitwise, memory addresses)
- Unix timestamps
- Base conversions
- Quick math under kodning

**Secondary:**
- CS-studenter
- DevOps/sysadmins
- Tekniskt intresserade power users

## Key HMW Questions
1. HMW göra en kalkylator som folk väljer över Google/Spotlight?
2. HMW göra history användbar, inte bara en lista?
3. HMW balansera power features med clean UI?
4. HMW göra keyboard navigation snabbare än mouse?
5. HMW visa alla baser samtidigt utan clutter?

## Explored Alternatives
**Crazy 8s utforskade:**
- Standard scientific calc (för generiskt)
- Financial calc (mättad marknad)
- Natural language calc (intressant men scope creep)
- Unit converter hybrid (nice-to-have, inte core)
- **Developer calc** (vald riktning - tydligt behov)
- Nisch-specifik (kök, gym) - kräver domänkunskap

**Mashups som inspirerade:**
- Calculator + Notion = beräkningar med kontext
- Calculator + Raycast = keyboard-first, instant results

## Competitive Landscape
| Konkurrent | Styrkor | Svagheter |
|------------|---------|-----------|
| BitCalculator (Android) | Gratis, alla baser | Endast mobil, basic UI |
| Programmer's Calculator (iOS) | Bitwise ops | iOS only, daterad design |
| calc.50x.ca (webb) | Klickbara bits | Minimal features |
| Windows Calculator (Programmer mode) | Inbyggd | Windows only, clunky |
| Soulver | Text + calc | Betalapp, inte dev-fokuserad |

**Öppning:** Modern webb-app med dev-features, keyboard-first, snygg design.

## Risks & Concerns (from Devil's Advocate)
| Risk | Mitigation |
|------|------------|
| "Alla har redan kalkylator" | Nisch-fokus på developers, inte alla |
| "Google löser det snabbare" | Persistent history, keyboard shortcuts, offline |
| "Mättad marknad" | Generell: ja. Modern dev-webb: nej |
| "Ingen laddar ner" | Webb-app = ingen nedladdning behövs |
| "Scope creep" | MVP: 3 core features, sen iterera |
